## Installation

YYCategoriesSub is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
source 'http://172.23.105.201:4080/cusc-ios-base-component/CUSpecs.git'

pod 'YYCategoriesSub'
```
